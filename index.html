Some Git Command


  • git config Usage: git config –global user.name “[name]” Usage: git config –global user.email “[email address]” This command sets the author name and email address respectively to be used with your commits

  • git init Usage: git init [repository name] This command is used to start a new repository.

  • git clone Usage: git clone [url] This command is used to obtain a repository from an existing URL.

  • git add Usage: git add [file] This command adds a file to the staging area.
